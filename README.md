## Deployment

How to install and run this code

```bash
  git clone https://github.com/rafiisite/bursa-kerja-khusu-TUGAS-AKHIR.git
```

```bash
  cd [folder name]
```

```bash
  composer install
```

```bash
  php artisan key:generate
```

```bash
  php artisan migrate
```

```bash
  php artisan serve
```

now this code is running in port : 8000
